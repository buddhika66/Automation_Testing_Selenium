# Automation Testing Project

This is a simple automation testing project created using Selenium.

## Tools Used
- Selenium WebDriver
- Java
- TestNG
- ChromeDriver
- Maven

## Test Scenario
Automate the search functionality of a google website.

## Steps Automated
1. Open browser
2. Navigate to website page
3. Type "Sri Lanka"
4. Click Enter
5. Verify search name successfully
